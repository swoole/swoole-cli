# 常见问题

## 快速安装 `PHP` 解释器

### Alpine
```bash
sh sapi/quickstart/linux/extra/alpine-php-init.sh
```

### Debian/Ubuntu
```bash
bash sapi/quickstart/linux/extra/debian-php-init.sh
```

## 安装 `Composer`

```bash
wget -O composer.phar https://mirrors.aliyun.com/composer/composer.phar
```
